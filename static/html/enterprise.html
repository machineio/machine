API Gateway

The API Gateway is a central hub to manage all the back-end setup, resources, configurations also handle security and limit quotes for all the account's

This RestFull API use all the methods defined in the http protocol

API Methods

OPTION

curl -X OPTIONS -H "Content-Type: application/json" https://api.cloudforest.ws/emails/ | jq

POST

curl -X POST -H "Content-Type: application/json" -d '{"account":"codemachine", "subject": "Email test", "source":"info@codemachine.io", "destination": "John johndoe@codemachine.io", "text":"this is a test email from the emails resource."}' https://api.cloudforest.ws/emails/ |jq

GET

curl -X GET -H "Content-Type: application/json" https://api.cloudforest.ws/tasks/bf5a694b-9e7f-4a68-8244-1d0c94ea4c1e |jq

PATCH

curl -X PATCH -H "Content-Type: application/json" -d '{"title": "please modify this title"}' https://api.cloudforest.ws/tasks/bf5a694b-9e7f-4a68-8244-1d0c94ea4c1e |jq
